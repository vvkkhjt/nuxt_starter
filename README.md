## Build Setup

``` bash
# 安装依赖
$ npm install # Or yarn install

# 开发端口，默认4000，需要修改到package.json文件内修改config字段
$ npm run dev

# 发布运行
$ npm run build
$ npm start

# 生成静态打包文件
$ npm run generate
```
